Instructions After Downloading

  1. Save the .zip file to your saves folder
    (on Windows, the address is %appdata%\StardewValley\Saves)
  2. Extract the files into a folder with the same name as the zip file (i.e. Maxan_199008108).
  3. Start Stardew Valley, and you should be able to see Goldrush Farm in your list of saved games!

If the download fails, try it again here.
